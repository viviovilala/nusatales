# Developer Skill Profile

Expected core capabilities for engineers working on NusaTales.

## Required Technical Skills

- Laravel API development with controllers, requests, resources, policies, and Eloquent.
- React JSX development with component composition, routing, and state management.
- REST API integration patterns between frontend and backend.
- Migration-based database evolution and relational modeling.
- Git-based collaboration with review-driven workflows.

## Backend Skill Expectations

- Build thin controllers and move business logic into services or actions.
- Design clean REST endpoints and consistent JSON responses.
- Work with authentication, authorization, caching, queues, and jobs.
- Write unit and feature tests for changed behavior.

## Frontend Skill Expectations

- Build reusable React components and page-level compositions.
- Manage local and shared state predictably.
- Handle loading, empty, success, and error states explicitly.
- Integrate authenticated API requests safely.

## Domain Awareness

Developers should understand the product scope:

- `NusaMandala` for cultural stories and mapping.
- `NusaLanglang` for exploration and itinerary-related features.
- `NusaKarya` for creator and monetization workflows.
- `Community` for social interaction and moderation.

## Delivery Checklist

- [ ] Can implement Laravel REST endpoints cleanly.
- [ ] Can structure React pages and components for scale.
- [ ] Can work with migrations, validation, and policies.
- [ ] Can debug local environment and deployment issues.
- [ ] Can document architectural decisions in Markdown.
